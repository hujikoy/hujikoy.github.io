Resources

This is a Learning / Help list of some intresting resources.

LLVM

  • Writting an LLVM Pass: 101 (pdf), a tutorial for LLVM in 2019
  • 2019 LLVM Developers' Meeting (site), which contains some tutorials and new technics
  • Getting Started With LLVM: Basics (video), good start for LLVM
  • llvm-tutor (project), a collection of self-contained reference LLVM passes. It's a tutorial that targets novice and aspiring LLVM developers
  • CS153:Compilers site, a compiler course useful for Static Analysis

Static Analysis

  • Dataflow Analysis (video), thorough introduction to dataflow analysis
    • Reaching Definition
    • Very Busy Expression
    • Available Expression
    • Live Variabless)
  • Data Flow Analysis (pdf), slides fot dataflow analysis
  • Data-Flow Analysis Programming Assignment (pdf), a lab exercise of dataflow analysis using LLVM (some good hints inside)

OS

Linux Kernel Memory Management

  • Linux内存管理slub分配器 (blog), a thorough introduction to slab&slub
  • Bochspwn Reloaded, slides for the bochspwn-reloaded (github repo, a tool to detect memory double-fetch, using bochs to taint memory)
  • What every programmer should know about memory, Main Memory(part 1), CPU caches(part 2)

Performance

  • io_uring 的介绍 - Lord of the io_uring

Cloud Native

  • LibOS: Library OS is the New Container - Graphene, pdf
  • Unikernel
    • Unikernels: The Rise of the Virtual Library Operating System, pdf
    • All About Unikernels
      • Part 1, What They Are, What They Do, and What’s New, site
      • Part 2, Two Different Approaches, MirageOS and Rumprun, site

Bugs

  • Dirty COW Demo, an intuitive visualization of dirty COW bugs

Security

  • Google Project Zero, News and updates from the Project Zero team at Google

Other

  • COMS W4995: Secure Software Development: Theory and Practice, course for existing tools and techniques for secure software development
